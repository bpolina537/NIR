# Научно-исследовательская работа

Учебный проект для производственной практики (НИР).

Тема: «Автоматизация учета и продаж товаров магазина».

Объект автоматизации — магазин товаров для дома и декора «Атмосфера».

## Планируемые модули

- адаптивная главная страница, каталог, карточка товара и оформление заказа;
- учет товаров, остатков, заказов и продаж;
- задания JavaScript из индивидуального задания;
- PHP-счётчик посещений, гостевая книга и калькулятор доставки;
- REST API товаров;
- база данных и запуск через Docker;
- контроль версий Git.

Подробная инструкция запуска будет добавлена после настройки окружения.
