## О проекте

«Атмосфера» — учебный интернет-магазин товаров для дома и декора. Проект объединяет клиентскую часть сайта, серверную обработку данных и средства автоматизации учета товаров, заказов и продаж.

Система предназначена для демонстрации навыков адаптивной верстки, программирования на JavaScript и PHP, работы с SQL, внешними сервисами, REST API и системой контроля версий Git. Проект выполняется в учебном объеме и не является коммерческим релизом.

## Основные страницы магазина

- **Главная страница** — информация о магазине, подборки товаров, преимущества и акции.
- **Каталог** — список товаров с ценами, остатками и средствами фильтрации.
- **Карточка товара** — фотография, описание, цена, характеристики и возможность оформить заказ.
- **Оформление заказа** — контактные данные покупателя, адрес доставки, карта и проверка формы.

## Автоматизируемые процессы

- хранение и получение сведений о товарах;
- учет цен, количества и товарных остатков;
- расчет итоговой стоимости товарных позиций;
- фильтрация и поиск товаров;
- оформление и сохранение заказов;
- расчет стоимости доставки;
- учет посещений сайта;
- хранение сообщений гостевой книги;
- предоставление доступа к товарам через REST API.
